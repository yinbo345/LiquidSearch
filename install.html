LiquidSearch 安装说明

免费安装，无需 Chrome 应用商店，3 分钟搞定 ✨

1

下载扩展文件

点击下方按钮下载 LiquidSearch 最新版 zip 压缩包，保存到本地任意位置（记住保存位置）。

📦 下载 ZIP 格式 🗜️ 下载 7Z 格式

⚠️ CRX 格式不再更新，请优先使用 ZIP 格式安装

💡 如果 GitHub 下载慢，也可以用国内镜像或 Gitee 镜像加速下载。

2

解压 zip 文件

找到下载好的 LiquidSearch_v1.2.zip，右键 → 解压到当前文件夹。
解压后得到一个 LiquidSearchEnhancer 文件夹，记住这个文件夹的位置。

⚠️ 注意：必须将 zip 解压，不能直接拖进浏览器！

🗜️ 我没有解压工具？获取 7-Zip
3

打开 Chrome 扩展管理页面

在 Chrome 地址栏输入以下地址并回车：

chrome://extensions/

或者在 Chrome 菜单 → 更多工具 → 扩展程序 打开。

4

开启「开发者模式」

在扩展管理页面的右上角，找到 开发者模式 开关，点击把它打开（变成蓝色/开启状态）。

💡 开启后页面左上角会出现「加载已解压的扩展程序」按钮。

5

加载扩展文件夹

点击左上角的 加载已解压的扩展程序 按钮，在弹出的窗口中选择第 2 步解压出来的 LiquidSearchEnhancer 文件夹，点击「选择文件夹」。

✅ 加载成功后，LiquidSearch 会出现在扩展列表里，右上角地址栏旁边也会出现 🔍 图标！

支持的浏览器
✅ Chrome ✅ Edge ✅ Brave ✅ Opera ✅ 360 极速浏览器
⚠️ 暂不支持：360 搜索（www.so.com）和 360 导航（hao.360.com / hao.360.cn），将在后续版本中适配。
